Чтобы запустить программу требуются программы gcc и make. Для ОС windows необходимо в папке с проектом в коммандной строке написать make windows. Позже программу можно запустить из появившегося исполняемого файла "app.exe". Для ОС GNU/Linux — make posix. Позже запускается из исполняемого файла "app".
