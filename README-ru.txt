===Установка и запуск===
Чтобы запустить программу требуются программы g++ и make. Для ОС windows необходимо в папке с проектом в коммандной строке написать make windows. Позже программу можно запустить из появившегося исполняемого файла "app.exe". Для ОС GNU/Linux — make posix. Позже запускается из исполняемого файла "app".

===Описание настроек===
value - число, которым будет заполняться поле или вырезаться коридор
seed - семя генерации случайных значений
start x, start y - точка, откуда будет начинаться коридор лабиринта (обычно необходимо изменить на 1, 1, чтобы оставалась внешняя стена)
invert x, invert y - изменение направление лабиринта двоичного дерева (по умолчанию -- вправо вниз)
max - максимальное значение генератора случайных чисел
depth - глубина фрактального шума
noise amplitude - амплитуда отклонений
avg - примерное среднее значение

Алгоритмы генерации лабиринтов создают коридоры, но не стены, поэтому вначале будет необходимо всё заполнить единицами (стенами).
